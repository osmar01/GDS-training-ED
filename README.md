# Backend

## Tech

GDS uses a number of technology to work properly:

- NodeJS
- MySql

## Installation

- Requires Node.js v10+ to run.
- Requires Sequelize Command-Line Interface (CLI). 

Install the dependencies and start the server.

```sh
cd backend
npm run create-database
npm start
```
